Design Specification

Barwaaqo Skills Design System

Navy + yellow ed-tech identity for Somalia and the diaspora. All pages must use tokens from css/tokens.css — no one-off colors or spacing.

Font: Inter Max width: 1200px Style: Learnify-inspired Breakpoints: 1024 / 768 / 540px

Color palette

Primary surfaces use navy; yellow drives action. Light sections use #F8FAFC.

Typography

Inter at 400 for body, 600–700 for UI labels and headings, 800 for hero display.

Learn Anytime, Anywhere

Section heading (700)

Body copy uses --text-muted on light backgrounds for descriptions, course blurbs, and FAQ answers. Line height stays at 1.5–1.7 for readability.

Caption / meta — 0.8rem, muted

Buttons

Pill shape, 2px border, semibold 0.9rem. Yellow is the primary action on dark; navy on light.

Badges & cards

Course cards: white, 16px radius, soft shadow. Badges sit top-left on thumbnails.
